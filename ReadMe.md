Rosh Synthetics - Premium Artificial Turf & Landscaping

Welcome to the official repository for Rosh Synthetics, a modern web application and landing page designed to showcase professional synthetic turf installations, paving, and landscaping services in Cape Town, South Africa.

🚀 Features

Responsive Landing Page: Fully optimized for mobile, tablet, and desktop devices with Tailwind CSS.

Interactive Quote Estimator: Dynamic real-time calculation widget for estimating artificial grass material and base preparation costs in South African Rand (ZAR).

Direct WhatsApp Integration: Instant quote requests and chat functionality mapped directly to business numbers.

Project Gallery & Lightbox: Interactive portfolio previews supporting multi-image carousels and fullscreen modals.

Modern Build System: Powered by Vite and Tailwind CSS v4 for lightning-fast performance and optimal asset bundling.

🛠️ Getting Started & Local Development

Follow these steps to run the project locally on your machine:

1. Prerequisites

Ensure you have Node.js (v18 or higher) installed on your system.

2. Install Dependencies

Clone the repository and install the required npm packages:

git clone https://github.com/your-username/rosh-synthetics.git
cd rosh-synthetics
npm install


3. Run the Development Server

Start the local Vite development server with Hot Module Replacement (HMR):

npm run dev


Open your browser and navigate to http://localhost:5173.

📦 Building for Production

To create an optimized production build ready for deployment:

npm run build


The compiled files will be generated inside the dist/ directory.
